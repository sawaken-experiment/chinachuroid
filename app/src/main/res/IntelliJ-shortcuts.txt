- クイック・フィックス (Quick Fix)
    [Alt] + [Enter]
- 行入れ替え
    [Alt] + [Shift] + ↑ or ↓
- カレント行削除
    [Ctrl] + y
- カレント行複製
    [Ctrl] + d
- コメント化、解除 //...
    [Ctrl] + /
- ブロックコメント化、解除 /*...*/
    [Ctrl] + [Shift] + /
- ローカル変数の割り当て
    [Ctrl] + [Alt] + v ([Alt] + [Enter]でもできる)
- 置換
    [Ctrl] + r
- 名前変更
    [Shift] + F6
- コードフォーマット
    [Ctrl] + [Alt] + l
- インポート文の最適化
    [Ctrl] + [Alt] + o
- System.out.println 入力
    sout [Ctrl] + j
- mainメソッド 入力
    psvm入力後に [Tab] を押す
- 改行コードを変更するCRLF -> LF
    Ctrl + Shift + Aを押すと機能名から呼び出せる入力ウインドウが現れるので
    LFと入力してEnterで実行
- ファイル検索
    [Ctrl] + [Shift] + f
- ファイル名で絞り込み検索して開く
    [Ctrl] + [Shift] + n
- 呼び出し元を探す(呼び出し階層)
    [Ctrl] + [Alt] + h
- インタフェースから実装クラスを探す(クイック型階層)
    [Ctrl] + [Alt] + b
- 定義箇所へジャンブ
    [Ctrl] + b
- ジャンプして移動した履歴を戻る/進む
    [Ctrl] + [Alt] + ← or →
- スイッチャーでエディタタブを移動する
    [Ctrl] + [Tab]
- 最後の編集箇所に戻る
    [Ctrl] + [Shift] + Backspace
- 次のメソッド定義にジャンプする
    [Alt] + ↑ or ↓
- エディタタブの移動
    [Alt] + ← or →
- テストクラスにジャンプする
    [Ctrl] + [Shift] + t
- コード内のファイルパス(クラスパス)文字列からそのファイルを開く
    コード内に記述されているファイルパスにCtrlを押しながらカーソルを合わせるとジャンプできる
- アクティブのエディタタブを閉じる
    [Ctrl] + F4
- Ctrl+Alt+U でそのクラスの階層をUMLダイアグラムで表示してくれる。
